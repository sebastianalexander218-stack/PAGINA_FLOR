# 🛒 Abarrotes Flor - Página Web

Sistema web para la gestión de ventas, productos y clientes de una tienda de abarrotes. Desarrollado con Flask y almacenamiento en archivos JSON.

## 🚀 Características

- ✅ Autenticación de usuarios (admin: alex@123 / alex123)
- ✅ Gestión de productos (CRUD)
- ✅ Carrito de compras
- ✅ Registro de clientes
- ✅ Panel de administración (Dashboard)
- ✅ Gráficos de ventas
- ✅ Notificaciones en tiempo real

## 🛠️ Tecnologías

- Python 3.14
- Flask
- HTML + CSS + JavaScript
- Chart.js (gráficos)
- JSON (almacenamiento)
